# Qmsg 酱

### 配置示例

```json
{
  "QMSG_KEY": "",
  "QMSG_TYPE": "",
  "MERGE_PUSH": ""
}
```

|       Name       |   归属   | 属性 | 说明                                                                                           |
| :--------------: | :------: | :--: | :--------------------------------------------------------------------------------------------- |
|  _**QMSG_KEY**_  | Qmsg 酱  | 推送 | Qmsg 酱推送[官方文档](https://qmsg.zendee.cn/) ,填写 `KEY` 代码即可                            |
| _**QMSG_TYPE**_  | Qmsg 酱  | 推送 | Qmsg 酱推送[官方文档](https://qmsg.zendee.cn/) ,如果需要推送到群填写 `group`,其他的都推送到 QQ |
| _**MERGE_PUSH**_ | 合并推送 | 配置 | **true**: 将推送消息合并；**false**: 分开推送                                                  |
