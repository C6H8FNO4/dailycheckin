# 飞书

### 配置示例

```json
{
  "FSKEY": "",
  "MERGE_PUSH": ""
}
```

|       Name       |   归属   | 属性 | 说明                                                                                      |
| :--------------: | :------: | :--: | :---------------------------------------------------------------------------------------- |
|   _**FSKEY**_    |   飞书   | 推送 | `https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxx` **xxxxxx** 部分就是需要填写的 FSKEY |
| _**MERGE_PUSH**_ | 合并推送 | 配置 | **true**: 将推送消息合并；**false**: 分开推送                                             |
