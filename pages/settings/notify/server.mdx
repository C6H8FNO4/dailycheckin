# Server 酱

### 配置示例

```json
{
  "SCKEY": "",
  "MERGE_PUSH": ""
}
```

|       Name       |   归属    | 属性 | 说明                                                                |
| :--------------: | :-------: | :--: | :------------------------------------------------------------------ |
|   _**SCKEY**_    | server 酱 | 推送 | server 酱推送[官方文档](https://sc.ftqq.com) ,填写 `SCKEY` 代码即可 |
| _**MERGE_PUSH**_ | 合并推送  | 配置 | **true**: 将推送消息合并；**false**: 分开推送                       |
