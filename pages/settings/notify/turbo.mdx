# Server 酱 TURBO

### 配置示例

```json
{
  "SENDKEY": "",
  "MERGE_PUSH": ""
}
```

|       Name       |      归属       | 属性 | 说明                                                                                  |
| :--------------: | :-------------: | :--: | :------------------------------------------------------------------------------------ |
|  _**SENDKEY**_   | Server 酱 TURBO | 推送 | server 酱 TURBO 推送[官方文档](https://sct.ftqq.com/sendkey) ,填写 `SENDKEY` 代码即可 |
| _**MERGE_PUSH**_ |    合并推送     | 配置 | **true**: 将推送消息合并；**false**: 分开推送                                         |
