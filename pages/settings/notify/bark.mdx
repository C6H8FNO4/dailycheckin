# BARK

### 配置示例

```json
{
  "BARK_URL": "",
  "MERGE_PUSH": ""
}
```

|       Name       |   归属   | 属性 | 说明                                                                                                            |
| :--------------: | :------: | :--: | :-------------------------------------------------------------------------------------------------------------- |
|  _**BARK_URL**_  |   BARK   | 推送 | BARK 推送[使用](https://bark.day.app/#/) ,填写 `BARK_URL` 即可，例如: `https://api.day.app/DxHcxxxxxRxxxxxxcm/` |
| _**MERGE_PUSH**_ | 合并推送 | 配置 | **true**: 将推送消息合并；**false**: 分开推送                                                                   |
